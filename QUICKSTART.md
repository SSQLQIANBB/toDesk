# 🚀 Remote 快速入门指南

## 📦 安装依赖

### 1. 后端依赖
```bash
cd backend-koa
pnpm install
pnpm add uuid
pnpm add -D @types/uuid
```

### 2. 前端依赖
```bash
cd client-vue
pnpm install
```

## ▶️ 启动服务

### 启动后端 (终端1)
```bash
cd backend-koa
pnpm dev
```
✅ 后端运行在: `http://localhost:3000`

### 启动前端 (终端2)
```bash
cd client-vue
pnpm dev
```
✅ 前端运行在: `http://localhost:5173` (或其他端口)

## 🎮 使用步骤

1. **打开多个浏览器窗口** (测试需要至少2个用户)
   - 窗口1: `http://localhost:5173/remote`
   - 窗口2: `http://localhost:5173/remote` (隐身模式或其他浏览器)

2. **两个窗口都点击"登录"按钮**
   - 系统会自动分配用户ID
   - 可以看到对方出现在在线用户列表中

3. **开始聊天**
   - 在任一窗口选择对方用户
   - 输入消息并发送

4. **发起视频通话**
   - 点击工具栏的 📹 图标
   - 对方会收到来电提示
   - 接听后即可开始视频通话

5. **屏幕共享**
   - 点击工具栏的 🖥️ 图标
   - 选择要共享的屏幕/窗口
   - 对方接听后可看到您的屏幕

## ⚡ 核心功能

| 功能 | 图标 | 说明 |
|------|------|------|
| 视频通话 | 📹 | 双向音视频通话 |
| 屏幕共享 | 🖥️ | 实时共享屏幕内容 |
| 即时消息 | 💬 | 文字聊天 |
| 在线状态 | 🟢 | 实时显示在线用户 |

- ✅ 发起视频通话

## 🔍 故障排查

### 问题：无法连接服务器
```bash
# 检查后端是否启动
curl http://localhost:3000
# 应该返回: TO DESK SERVER
```

### 问题：看不到摄像头画面
- 检查浏览器权限（允许摄像头/麦克风访问）
- 使用 `http://localhost` 而非 `http://127.0.0.1`
- Chrome 需要安全上下文 (localhost 或 HTTPS)

### 问题：两个用户看不到对方
- 确保两个窗口都成功登录
- 检查浏览器控制台是否有错误
- 刷新页面重新登录

## 📱 浏览器要求

| 浏览器 | 版本 | 支持度 |
|--------|------|--------|
| Chrome | 最新 | ✅ 完全支持 |
| Firefox | 最新 | ✅ 完全支持 |
| Edge | 最新 | ✅ 完全支持 |
| Safari | 15+ | ⚠️ 部分功能需测试 |

## 💡 提示

1. **测试建议**: 使用两个不同的浏览器或隐身窗口测试
2. **性能**: 本地测试效果最佳，跨网络可能需要 TURN 服务器
3. **调试**: 打开浏览器控制台查看详细日志
4. **权限**: 首次使用需授权摄像头、麦克风和屏幕共享权限
