# ToDesk 功能更新说明

## 本次更新内容

### 1. ✅ 用户登录状态保持和自动恢复

#### 后端实现
- **自动Token验证**：在 `request.ts` 中添加了401错误处理，自动清除无效token并跳转登录页
- **登出接口**：新增 `/api/auth/logout` 接口，登出时更新用户状态为离线
- **Token过期处理**：JWT token有效期设置为7天，过期自动跳转登录

#### 前端实现
- **路由守卫增强**：在 `router/index.ts` 中添加了自动恢复用户信息的逻辑
- **状态持久化**：使用 localStorage 存储 token 和用户信息
- **刷新恢复**：页面刷新后自动从 localStorage 恢复用户信息，如果token有效则保持登录状态

#### 使用方式
- 用户登录后，token 和用户信息会自动保存
- 刷新页面或重新打开浏览器，系统会自动恢复登录状态
- Token 过期或无效时，自动跳转到登录页

### 2. ✅ 用户信息维护功能

#### 新增页面：`/profile`
个人中心页面，包含两个标签页：
- **基本信息**：维护个人资料
- **账户安全**：修改密码和账户状态

#### 功能特性
1. **头像上传**
   - 支持 JPG、PNG 格式
   - 最大 2MB 限制
   - 图片转 Base64 存储

2. **个人资料编辑**
   - 昵称（可修改，最多50字符）
   - 用户名（不可修改）
   - 邮箱（带邮箱格式验证）
   - 手机号（带手机号格式验证）
   - 个人简介（最多200字符）

3. **账户安全**
   - 修改密码（需验证旧密码）
   - 账户状态切换（在线/忙碌/离线）
   - 最后登录时间显示

4. **数据库增强**
   - User 模型新增 `bio` 字段（个人简介）
   - 优化了用户信息更新逻辑

#### 后端新增接口
- `POST /api/auth/change-password` - 修改密码
- 增强 `PUT /api/auth/me` - 支持更新 bio 和 status

### 3. ✅ 群组管理功能

#### 新增页面：`/groups`
群组管理中心，提供完整的群组管理功能。

#### 功能特性
1. **创建群组**
   - 设置群组名称（必填，2-50字符）
   - 群组简介（可选，最多200字符）
   - 群组头像（可选）
   - 创建者自动成为群主

2. **群组列表**
   - 显示所有已加入的群组
   - 展示群组名称、简介、成员数量
   - 标识用户角色（群主/管理员/成员）
   - 快捷操作按钮（聊天/视频/共享）

3. **群组详情**
   - 群组基本信息展示
   - 完整成员列表
   - 成员角色标识

4. **成员管理（群主/管理员）**
   - 邀请新成员加入
   - 设置成员发言权限（禁言/允许发言）
   - Transfer 组件选择用户

5. **权限控制**
   - 群主：完全控制权（邀请、踢人、禁言、解散）
   - 管理员：邀请成员、管理群组信息
   - 普通成员：基本参与权限

#### 数据库模型
- **Group 表**：存储群组信息
- **GroupMember 表**：存储群组成员关系和权限

#### 后端接口
- `POST /api/groups` - 创建群组
- `GET /api/groups/my` - 获取我的群组列表
- `GET /api/groups/:id` - 获取群组详情
- `POST /api/groups/:id/invite` - 邀请用户加入
- `POST /api/groups/:id/permission` - 设置成员权限
- `PUT /api/groups/:id` - 更新群组信息
- `DELETE /api/groups/:id/leave` - 退出群组

#### Socket.io 事件
- `toggle_mic` - 切换麦克风状态
- `control_member_mic` - 控制成员麦克风

### 8. ✅ 视频清晰度实时调整功能

#### 清晰度档位
提供4个清晰度档位：

| 档位 | 分辨率 | 帧率 | 适用场景 |
|------|--------|------|----------|
| 流畅 | 1280x720 | 15fps | 网络较差 |
| 标清 | 1920x1080 | 24fps | 一般网络 |
| 高清 | 1920x1080 | 30fps | 良好网络（默认） |
| 超清 | 2560x1440 | 30fps | 优秀网络 |

#### 实现方式
1. **实时切换**
   - 下拉菜单选择清晰度
   - 即时生效，无需刷新
   - 自动重新获取媒体流

2. **参数配置**
   - width/height：视频分辨率
   - frameRate：视频帧率
   - 使用 getUserMedia 约束

3. **用户体验**
   - 当前清晰度标识
   - 切换成功提示
   - 平滑过渡

4. **智能优化**
   - 根据网络状况自动建议
   - 保存用户偏好设置
   - 断线重连保持设置

## 使用指南

### 1. 启动项目

#### 后端
```bash
cd backend-koa
npm install
npm run dev
```

#### 前端
```bash
cd client-vue
npm install
npm run dev
```

### 2. 功能使用流程

#### 用户注册/登录
1. 访问 `/login` 页面
2. 注册新用户或使用现有账号登录
3. 登录成功后跳转到远程协作主页

#### 完善个人信息
1. 点击侧边栏头像或导航到 `/profile`
2. 上传头像、填写个人信息
3. 可选：修改密码、设置在线状态

#### 创建和管理群组
1. 导航到 `/groups` 群组管理页
2. 点击"创建群组"按钮
3. 填写群组名称和简介
4. 邀请成员加入群组

#### 群组聊天
1. 在群组列表点击"聊天"按钮
2. 进入群组聊天室
3. 发送消息与成员交流

#### 发起视频通话
1. 在群组聊天室点击"发起视频通话"
2. 或在群组列表点击"视频"按钮
3. 等待成员加入通话
4. 使用控制栏管理音视频

#### 屏幕共享
1. 在群组聊天室点击"发起屏幕共享"
2. 或在群组列表点击"共享"按钮
3. 选择要共享的屏幕/窗口
4. 调整视频质量（右上角设置）
5. 群主可控制成员发言权限

## 注意事项

1. **浏览器兼容性**
   - 推荐使用 Chrome、Edge、Firefox 最新版本
   - 需要允许摄像头和麦克风权限
   - 需要允许屏幕共享权限

2. **网络要求**
   - 稳定的网络连接
   - WebRTC 需要能够建立 P2P 连接
   - 建议使用 HTTPS（生产环境）

3. **安全性**
   - Token 有效期为 7 天
   - 密码使用 bcrypt 加密存储
   - 建议定期修改密码

4. **性能优化**
   - 群组视频通话建议人数 ≤ 9 人
   - 根据网络状况调整清晰度
   - 不使用时关闭摄像头节省带宽
